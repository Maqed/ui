---
title: Introduction
description: Herocn is a set of customizable, accessible components heavily inspired by HeroUI.
---

Ever tried using [HeroUI v3](https://v3.heroui.com/) but found it hard to integrate with [shadcn/ui](https://ui.shadcn.com/) registry? This is what **Herocn** is for.

We give you **HeroUI** design system but with **shadcn**'s abstraction, where you can use **shadcn**'s registry without the hassle of having to rewrite it to **HeroUI**'s abstraction.
